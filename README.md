# SCMS Delivery History Dataset Analysis


## Project Overview
This project focuses on analyzing the SCMS (Supply Chain Management System) Delivery History Dataset using Python. The objective of this analysis is to explore shipment operations, delivery performance, freight costs, and logistics trends through data cleaning, preprocessing, visualization, and exploratory data analysis.

---

## Problem Statement
Efficient supply chain management plays a critical role in logistics and delivery operations. This project aims to identify shipment patterns, delivery delays, cost distribution, and operational insights that can help improve logistics performance and decision-making.

---

## Project Workflow
### 1. Data Collection
- Imported and explored the SCMS Delivery History Dataset.

### 2. Data Cleaning & Preprocessing
- Handled missing values
- Removed duplicates
- Converted data types
- Treated inconsistent records

### 3. Exploratory Data Analysis (EDA)
- Shipment analysis
- Freight cost analysis
- Delivery trend analysis
- Country-wise and vendor-wise analysis

### 4. Data Visualization
Created multiple visualizations to understand:
- Shipment distribution
- Delivery delays
- Freight cost patterns
- Logistics performance trends

---

## Key Insights
- Identified major shipment and delivery trends.
- Analyzed freight cost distribution across shipments.
- Evaluated delivery performance using visual analysis.
- Detected operational patterns useful for logistics optimization.

---

## Conclusion
The project demonstrates how Python-based data analysis techniques can be used to extract meaningful business insights from logistics and supply chain datasets. The analysis helps in understanding shipment performance, operational efficiency, and delivery-related trends.
